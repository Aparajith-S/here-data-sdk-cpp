Detailed Description

The DataRequest class encapsulates the fields required to request the data for a given catalog and layer and partition (aka tile).

Layer Id is required, and either Partition Id or Data Handle must be provided. If both Partition Id and Data Handle are set in the request, the Request fails with an ErrorCode::InvalidRequest Error.
